TCP Pathology has moved to Infinity X software for placing service requests


TCP Pathology is now using Infinity X through Sinai Health Research for all service requests.


New service requests will only be accepted through Infinity X. Submission of new service requests to TCP Pathology will no longer be possible through TCP LIMS.


Access the new system here: https://infinity.ideaelan.com/sinai

HOW TO ACCESS THE PATHOLOGY CORE

  1. Existing Infinity X and Pathology Users
    • Request access to the pathology facility & place a service request

  2. Existing Pathology User, new to Infinity X

      First, check if your lab is on Infinity X.

    • No → contact Freda Qi (fqi@lunenfeld.ca) to add your lab
    • Yes → request access to your lab if not preassigned

      • Request access to the pathology facility & place a service request


  3. New Pathology User

      Welcome! We’re excited to work with you.

    • Contact Facility Manager Vivian Bradaschia (bradaschia@lunenfeld.ca) to discuss your project
    • Then follow the steps above to sign onto Infinity X

SIGNING ONTO INFINITY X

  • lunenfeld.ca or sinaihealth.ca users: sign on using your Single-Sign On (SSO)
  • external users: sign on using your Institute email and password
